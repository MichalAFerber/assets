# _Inbox — drop unsorted assets here

This is a staging area. Copy any new assets (wallpapers, icons, logos,
avatars, etc.) into this folder in any mess, then ask Claude:

> organize and dedupe the _Inbox

Claude will hash, deduplicate, categorize, and move each file into the right
place in the vault, then delete the empty inbox contents. See
[`CLAUDE.md`](../CLAUDE.md) for the exact rules it follows.

**Everything in here except this README is git-ignored** — inbox files never
get committed from the inbox itself; they only enter the repo once sorted into
a real category folder. The underscore prefix also keeps the folder off the
published site.
